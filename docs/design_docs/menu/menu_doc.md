# メニュー画面

![メニュー画面](menu_design.svg)

## 画面設計
1. ユーザー名入力欄
* 半角英数字記号5文字以上50桁以内入力可。
* 上記以外の場合以下のエラー文を入力欄下に表示する。
　"半角英数字・記号を5文字以上50桁以内で入力してください。"

2. パスワード入力欄
* 半角英数字記号8文字以上255桁以内入力可。
* 上記以外の場合以下のエラー文を入力欄下に表示する。
　"半角英数字・記号を8文字以上255桁以内で入力してください。"

3. ログオンボタン
* ユーザー名とパスワードが正しい状態でクリックするとメニュー画面に遷移する。
* ユーザー名とパスワードが誤った状態でクリックすると以下のエラー文がボタン下に表示される。
　"ユーザー名かパスワードが正しくありません。"

4. 新規登録ボタン
* クリックすると新規登録画面に遷移する。
