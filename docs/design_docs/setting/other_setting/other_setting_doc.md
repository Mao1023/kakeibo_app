# その他設定画面

![その他設定画面](other_setting_design.svg)

## 画面設計
1. ログオフボタン
* クリックするとログオフ画面に遷移する。

2. 月の開始日入力欄
* ブランク不可。
* 初期表示は1が入力された状態で表示される。

3. 月の開始額入力欄
* ブランク不可。
* 初期表示は0が入力された状態で表示される。

4. 先月分の繰り越し設定ボタン
* トグルボタン。
* 初期表示はオフの状態で表示される。

6. 登録ボタン
* 各入力欄に正常に値が入力されている場合、クリック可。
* クリックすると登録が完了する。
* 上記以外の場合、クリック不可。

7. 戻るボタン
* クリックするとメニュー画面に遷移する。
