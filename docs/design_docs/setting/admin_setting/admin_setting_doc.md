# 管理者画面

![管理者画面](admin_setting_design.svg)

## 画面設計
1. ログオフボタン
* クリックするとログオフ画面に遷移する。

2. ユーザー名一覧
* ユーザー番号、ユーザー名、管理者権限の有無を表にまとめる。
* 編集ボタンをクリックすると右の編集欄に登録内容が表示され、編集、更新ができる。
* 削除ボタンをクリックすると該当する項目を削除する。
* 現在ログオン中のユーザーはグレーアウトされ、編集、削除は不可。
* カーソルを動かすことで表示しきれない項目を閲覧する。

4. 管理者権限設定ボタン
* トグルボタン。
* 初期表示はオフの状態で表示される。

6. 更新ボタン
* クリックすると登録が完了する。

7. 戻るボタン
* クリックするとメニュー画面に遷移する。
