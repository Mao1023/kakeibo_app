# 支出入力モーダル

![支出入力モーダル](kakeibo_shisyutsu_design.svg)

## 画面設計
1. 支出年月日入力欄
* 家計簿画面のカレンダーの日付をクリックするとその年月日が入力される
* 入力欄をクリックするとカレンダーが表示され、年月日を編集できる。

2. 項目名入力欄
* セレクトボックス。
* 初期表示はログオン中のユーザーが支出項目として設定した項目の中で最もIDが若いものが選択された状態で表示される。

3. 支出分類ボタン
* 選択必須。
* 択一式。

4. 支出名入力欄
* ブランク不可。
* 100桁以内入力可。
* 上記以外の場合以下のエラー文を入力欄下に表示する。  
    "1文字以上100桁以内で入力してください。"

5. 支出額入力欄
* ブランク不可。
* 上記以外の場合以下のエラー文を入力欄下に表示する。  
    "支出額を入力してください。"

6. 登録ボタン
* 各入力欄に正常に値が入力されている、かつ支出分類が選択されている場合、クリック可。
* クリックすると登録が完了し、支出入力モーダルを閉じる。
* 上記以外の場合、クリック不可。

7. 戻るボタン
* クリックするとメニュー画面に遷移する。
