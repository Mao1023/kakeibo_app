# 家計簿画面

![家計簿画面](kakeibo_design.svg)

## 画面設計
1. ログオフボタン
* クリックするとログオフ画面に遷移する。

2. 今月分開始額入力欄
* ブランク不可。
* 上記以外の場合以下のエラー文を入力欄下に表示する。  
    "今月分開始額を入力してください。"
* その他設定画面の月の開始額入力欄で設定した値が表示される。
* 入力欄をクリックすると値の編集ができる。


3. 固定費額一覧
* ブランク不可。
* 上記以外の場合以下のエラー文を一覧下に表示する。  
    "固定費額を入力してください。"
* 入力可。
* 項目設定画面の固定費金額入力欄で設定した値が表示される。
* 入力欄をクリックすると値の編集ができる。

4. 生活費残高入力欄
* ブランク不可。
* 上記以外の場合以下のエラー文を一覧下に表示する。  
    "生活費残高を入力してください。"
* 入力可。
* 初期表示は0が入力された状態で表示される。

5. 自動計算ボタン
* クリックすると今月分開始額（繰越代＋今月分開始額）から固定費合計が引かれた値を自動計算し、生活費残高に表示する。

6. 支出入力ボタン
* クリックすると支出入力モーダルだ表示される。

7. 家計簿一覧
* 日付、項目設定画面で設定した項目のうち支出に分類された各項目名、日合計、月累計、残高、残高/残り日数を表にまとめる。
* 日付のはじめはその他設定画面の月の開始日入力欄で設定した日付とする。
* 日合計はその日の使用額を合計した値を表示する。
* 月累計はその日までの使用額を累計した値を表示する。
* 残高は生活費残高からその日までの使用額を差し引いた値を表示する。
* 残高/残り日数は残高を残り日数で割った値を表示する。

8. 戻るボタン
* クリックするとメニュー画面に遷移する。
