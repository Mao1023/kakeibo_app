# 新規登録画面

![新規登録画面](new_design.svg)

## 画面設計
1. ユーザー名入力欄
* ブランク不可。
* 半角英数字記号5文字以上50桁以内入力可。
* 上記以外の場合、以下のエラー文を入力欄下に表示する。
　"半角英数字・記号を5文字以上50桁以内で入力してください。"
* 既存のユーザー名が使用されている場合、以下のエラー文を入力欄下に表示する。
　"既に使われているユーザー名が登録されています。異なるユーザー名を入力してください。"

2. パスワード入力欄
* ブランク不可。
* 半角英数字記号8文字以上255桁以内入力可。
* 上記以外の場合、以下のエラー文を入力欄下に表示する。
　"半角英数字・記号を8文字以上255桁以内で入力してください。"

3. パスワード(確認用)入力欄
* ブランク不可。
* 半角英数字記号8文字以上255桁以内入力可。
* 上記以外の場合、以下のエラー文を入力欄下に表示する。
　"半角英数字・記号を8文字以上255桁以内で入力してください。"
* 入力された文字列がパスワード入力欄に入力された文字列と異なった場合、以下のエラー文を入力欄下に表示する。
　"パスワード入力欄に入力されたパスワードと異なります。"

4. 戻るボタン
* クリックすると登録をせずログオン画面に遷移する。

5. 登録ボタン
* 正常に値が入力された状態でクリックすると、ユーザー登録が完了しログオン画面に遷移する。
