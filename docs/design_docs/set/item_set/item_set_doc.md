# 設定画面

![設定画面](item_set_design.svg)

## 画面設計
1. ログオフボタン
* クリックするとログオフ画面に遷移する。

2. 項目名一覧
* 項目番号、項目名、分類、固定費金額を表にまとめる。
* 編集ボタンをクリックすると右の登録欄に登録内容が表示され、編集、更新ができる。
* 削除ボタンをクリックすると該当する項目を削除する。
* カーソルっを動かすことで表示しきれない項目を閲覧する。

3. 項目名入力欄
* ブランク不可。
* 50桁以内入力可。
* 上記以外の場合以下のエラー文を入力欄下に表示する。  
    "50桁以内で入力してください。"

4. 分類ボタン
* 択一式。

5. 固定費金額入力欄
* 分類で`固定費`が選択された場合、表示される。
* ブランク不可。
* 初期表示は0が入力された状態で表示される。

6. 登録ボタン
* 各入力欄に正常に値が入力されている、かつ分類が選択されている場合、クリック可。
* クリックすると登録が完了し、項目名一覧が自動更新される。
* 上記以外の場合、クリック不可。

7. 戻るボタン
* クリックするとメニュー画面に遷移する。
